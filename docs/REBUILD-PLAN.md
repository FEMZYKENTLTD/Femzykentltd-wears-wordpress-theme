## Theme/template approach chosen
A lightweight custom WordPress theme was prepared as a **landing-first theme**.

### Why this approach
- uploadable and activatable as a theme
- no page builder dependency required
- easier to keep fast and premium-looking
- editable from WordPress admin
- safe base for later WooCommerce mapping

---

## What the theme includes
- custom **Shoes** post type
- built-in **Orders** post type for order submissions
- front-page landing design
- thank-you page template
- editable theme settings under **Appearance → Femzykentltd Wears Settings**
- seeded 38 demo shoe slots
- policy/contact pages auto-created on theme activation

---

## How the 38 shoes will work
Each shoe slot can be updated in **WordPress Admin → Shoes**:
- title
- featured image
- code
- current price
- old/reference price
- sizes
- short description
- badge (optional)

When you send the real images, we can swap them in and tune:
- product names
- price points
- display order
- best-seller badges
- campaign copy
