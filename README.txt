Femzykentltd Wears Landing Theme
=========================

What this theme does
--------------------
- Creates a premium landing-page home page for Femzykentltd Wears.
- Creates a Shoes content type for your 38 shoes.
- Creates an Orders content type for submitted orders.
- Creates Home, Thank You, Privacy Policy, Terms, Delivery & Returns, and Contact pages on activation.
- Sets Home as the static front page automatically.

Setup after activation
----------------------
1. Go to Appearance > Femzykentltd Wears Settings.
2. Update brand name, support phone, WhatsApp link, thank-you WhatsApp URL, email, and copy.
3. Go to Shoes in the admin menu.
4. Edit the 38 demo shoe entries.
5. The theme already includes 38 bundled demo product images as fallbacks.
6. Optionally replace any bundled image by setting a Featured Image on that shoe post.
7. Update codes, prices, old prices, and short descriptions.
8. Test the order form.

Order flow
----------
- Product button fills the shoe code in the order form.
- Shoe codes are plain numbers (for example: 1, 2, 26, 38).
- Quantity accepts any custom value from 1 upward.
- Form submissions are stored in Orders and emailed to the admin/support email.
- Order details, including delivery address, can be viewed and edited inside WordPress Admin > Orders.
- Successful submissions redirect to the Thank You page.

Later upgrade path
------------------
This theme is intentionally landing-first. It can later be mapped into WooCommerce without redesigning the brand direction.
